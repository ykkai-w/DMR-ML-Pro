# DMR-ML Pro 快速开始

## 首次使用

### 1. 安装Python（5分钟）

**下载地址：** https://www.python.org/downloads/

**Windows：**
- 下载后双击安装
- ⚠️ **必须勾选** "Add Python to PATH"
- 点击"Install Now"

**macOS：**
- 下载 `.pkg` 文件
- 双击安装

---

### 2. 配置Token（2分钟）

1. 注册Tushare：https://tushare.pro/register
2. 复制Token
3. 打开 `config.py`，找到第9行：
   ```python
   TUSHARE_TOKEN = ""  # 填入你的Token
   ```

---

### 3. 启动系统

**Windows：** 双击 `🚀DMR-ML-Pro启动器.bat`  
**macOS：** 双击 `🚀DMR-ML-Pro启动器.command`

首次启动会自动安装依赖（1-2分钟），浏览器会自动打开。

---

## 日常使用

每次使用只需双击启动器，浏览器自动打开。

**注意：** 不要关闭黑色命令行窗口，关闭 = 停止服务。

---

## 常见问题

**macOS双击没反应？**  
右键 → "打开" → "打开"

**浏览器没自动打开？**  
手动访问：http://localhost:8501

**端口被占用？**  
关闭其他DMR-ML Pro或重启电脑

**依赖安装失败？**  
检查网络，或手动执行：`pip install -r requirements.txt`
